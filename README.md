# Project Management Toolkit

A lightweight collection of practical templates for managing projects of any size.

This repository contains reusable documents for planning, tracking, and delivering projects efficiently.

## Contents

- Project Lifecycle
- Risk Register
- Meeting Notes Template
- Task Prioritization
- Change Request Template
- Stakeholder Matrix

## Who is this for?

- Project Managers
- Startup Founders
- Product Managers
- Team Leads
- Students

## Goals

- Standardize project documentation
- Improve communication
- Track risks early
- Prioritize work effectively
- Manage stakeholders
